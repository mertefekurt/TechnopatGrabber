# Technopat Grabber

![Technopat Grabber cover](assets/readme-cover.svg)

A small terminal reader for Technopat headlines. It fetches the latest stories, lets you move through the feed, and opens the article detail when you want to read more.

## Run

```bash
git clone https://github.com/mertefekurt/TechnopatGrabber.git
cd TechnopatGrabber
python -m pip install -r requirements.txt
python3 main.py
```

## Workflow

- Fetch current headlines
- Navigate with simple keyboard commands
- Open article details on demand
- Keep the reading flow in the terminal
